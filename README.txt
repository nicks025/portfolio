# Portfolio Contact Form Setup

## Frontend (wuaze.com)
1. Upload the contents of the 'frontend' folder to your Wuaze hosting.

## Backend (Render)
1. Go to https://render.com and create a free account.
2. Create a new Web Service.
3. Connect your backend folder from GitHub or upload manually.
4. Set build command: npm install
5. Set start command: npm start
6. Add Environment Variables if needed.
7. Deploy and get your Render backend URL (e.g., https://your-app.onrender.com).

## Update Frontend
1. In frontend/script.js, replace `https://YOUR_RENDER_BACKEND_URL/send` with your Render backend URL.
